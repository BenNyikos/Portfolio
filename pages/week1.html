Module summary

During the first module we went through the introduction, how the class gonna look like, etc.
We also learned about how internet works in basic terms e.g.:

  • IP addresses, like Localhost(127.0.0.1), Google DNS, Internal IP
  • Ports, which are identify specific processes or types of network services
  • Protocols, like HTTP, HTTPS, DNS, etc.
  • HTTP = HyperText Tranfer Protocol communication between user and servers
  • HTTP response codes, like 404: Not found, 500: Server error
  • Difference between front- and back-end

Learned a bit about HTML

  • What is HTML = Hyper Text Markup Language
  • What are HTML tags = headers, divisions, lists, paragraphs, etc.
  • What are HTML attributes = type, name, size, lang, etc.
